Registration
Input: Full Name , Email / Phone, Password, (OTP Verification)
Action: After Successful Registration redirect to the login page

Login
Input: Email / Phone, Password
Rules: 
A user can only log in to one device at a time; multiple devices cannot be active simultaneously
First login from a new device or IP address → OTP verification required
Same device + same IP → OTP not required 
After login, user is redirected to the relevant service dashboard

Identity Verification (KYC)
Full Name, Photo (KYC), Gender, Date of Birth, Nationality, Government ID (NID/Passport/Driving License) (KYC), Passport Number, Permanent Address (Country, City, State, ZIP Code, Address)
Rules: 
This applies when more than 70 percent of the total project cost is outstanding
If you suspect fake activity
If you transact large amounts of money
If you see suspicious transactions
If you detect suspicious activity
KYC is required before withdrawing money
Unusual login / transaction
If you deposit / withdraw more than a certain limit
If you receive a high-value payment
If multiple transactions are fast

Change password
Input: Current Password, New Password, Confirm Password 
Rules:
After a successful password change, the user will be automatically logged out and must log in again

Account Security (2FA)
Input: Select one of the previously added and verified email addresses or phone numbers for OTP delivery 
Primary Email or Phone + Password → OTP Verification required 
Rules:
OTP will also be required when logging in from a Login with a new device, Delete cookies/cache, Don't enable "Remember this device", Change VPN/IP/location, Change password, Suspicious activity detected, Attempt to login from a different country,

Account Delete
Rules:
Wallet balance must be $0.00 
Remaining unpaid project amount must be $0.00 
No active or ongoing projects 
If all conditions are met, the account is deleted immediately and status set to Dormant

Account Recovery
Rules:
A deleted account can be restored within 60 days 
After 60 days, the account cannot be recovered

Forgot Password
Input: Primary or Secondary Email / Phone (OTP Verification)
Input: New Password  ,Confirm Password 
Rules:
If a secondary email or phone is used for recovery, it will be set as the new primary 
Two-Factor Authentication (2FA) will be automatically turned OFF after password recovery
